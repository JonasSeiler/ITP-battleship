repository for a battleship project written in Java. 

file structure should be something like that: 

ITP-battleship/  
├── src/  
│   ├── .java  
│   ├── ...  
│   └── .java  
├── img/  
│   ├── .jpeg  
│   ├── ...  
│   └── .png  
├── .jar  
├── battleshipClass.uxf  
└── README.md  

The class diagramm is stored in the .uxf file (battleshipClass.uxf) and can be 
opened via the application UMLet or in the browser via this URL:
https://www.umletino.com
